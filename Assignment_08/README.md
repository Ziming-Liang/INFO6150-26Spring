
## Setup
1. Install dependencies: `npm install`
2. Start MongoDB: `mongod`
3. Start server: `node server.js`

## API Endpoints
- POST /user/create
- PUT /user/edit
- GET /user/getAll
- DELETE /user/delete
- POST /user/uploadImage

## Testing
Import `Assignment8_Postman_Collection.json` into Postman
