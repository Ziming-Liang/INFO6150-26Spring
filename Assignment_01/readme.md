1. **Favicon** - `<link rel="icon">` - Website icon in browser tab

2. **Semantic Elements:**
   - `<header>` - Page header
   - `<section>` - Content sections
   - `<article>` - Article content
   - `<aside>` - Sidebar
   - `<footer>` - Page footer

3. **Table** - `<table>`, `<tr>`, `<th>`, `<td>` - Course comparison table

4. **Form:**
   - `<form>` - Registration form
   - `<input type="text">` - Text input
   - `<input type="email">` - Email input
   - `<input type="password">` - Password input
   - `<datalist>` - Dropdown suggestions
   - `<label>` - Input labels

5. **Images:**
   - `<img>` - Images
   - `<figure>` - Image container
   - `<figcaption>` - Image captions

6. **Hyperlink** - `<a>` - Links

7. **Button** - `<button>` - Form buttons

8. **Audio** - `<audio>` - Audio player

9. **Video** - `<video>` - Video player

10. **Details and Summary** - `<details>`, `<summary>` - Expandable content

11. **Contact Links:**
    - `<a href="mailto:">` - Email link
    - `<a href="tel:">` - Phone link

## Files
- index.html
- styles.css (external CSS)

- images/ (3 images)
