## Description
This project implements a form validation system using HTML, CSS, and JavaScript.  
The form includes real-time validation, dynamic fields, and displays submitted data in a table.

## Features
- Real-time validation using JavaScript
- Email validation restricted to @northeastern.edu
- Phone number validation with input masking: (XXX) XXX-XXXX
- Zip code validation (5 digits)
- Submit button enabled only when all fields are valid
- Dropdown list with dynamic checkbox
- Conditional text field based on checkbox selection
- Display submitted data in a table below the form
- Street Address 2 is optional with live character counter
